# zomato-powerbi-dashboard

# Project Overview
This project analyzes Zomato restaurant data to uncover insights about restaurant distribution, customer preferences, pricing trends, and service availability.
The goal is to help understand market trends and support data-driven decision making in the food industry.

# Dataset
- Source: Zomato dataset
- Total Records: 9000+ restaurants
- Features: Restaurant name, location, rating, cost, cuisine, votes, online delivery, table booking,price_bucket,rating_bucket

# Tools & Technologies
- Power BI
- Power Query (Data Cleaning & Transformation)
- DAX (Calculated Measures)
- Data Visualization

# Key Insights
- Restaurants are highly concentrated in metro cities, especially New Delhi
- India dominates the platform with 8652 restaurants
- Majority of restaurants fall under the “Good” rating category (48.06%)
- Budget segment dominates the market (81.03%)
- Service features like online delivery and table booking are underutilized

# Dashboard Features
- Designed a multi-page interactive dashboard with a navigation button to switch between overview and detailed analysis pages  
- Implemented dynamic slicers (Country, City, Cuisine, Price Range, Rating Bucket, Year) for real-time filtering  
- Developed KPI cards to highlight key metrics such as Total Restaurants, Average Rating, Total Votes, and Cost for Two  
- Visualized rating distribution using donut charts to categorize restaurants (Excellent, Good, Average, Poor)  
- Built comparative analysis of price segments vs average ratings to understand customer spending behavior  
- Analyzed service adoption using bar charts for Online Delivery and Table Booking availability  
- Created geographic and categorical insights using city-wise and country-wise restaurant distribution  
- Designed trend analysis using line charts to track restaurant growth over time  
- Highlighted top-performing cities and cuisines based on restaurant count and popularity  
- Applied consistent color theme and layout for better user experience and readability

# Conclusion

- Zomato operates across 15 countries and 141 cities, with a significant concentration in India, indicating a strong primary market presence.  
Restaurant distribution is highly skewed toward metro cities, highlighting urban market dominance.  
- The majority of restaurants fall within the budget segment, reflecting price-sensitive customer behavior.  
Ratings analysis suggests moderate service quality, indicating opportunities for improvement in customer experience.  
- Overall, these insights demonstrate how data-driven analysis can support strategic decisions to enhance service offerings, optimize pricing strategies, and drive business growth.
